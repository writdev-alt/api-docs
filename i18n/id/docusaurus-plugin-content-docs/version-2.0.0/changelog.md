## Changelog

## 2.0.0

- Menstandarkan format dokumentasi endpoint untuk bagian request/response.
- Menambahkan halaman khusus untuk:
  - Base URL dan Environment
  - Header
  - Rate Limit
  - Penanganan Error
  - Indeks endpoint API
  - SDK dan Contoh Kode
  - FAQ
- Menambahkan dan menyempurnakan dokumentasi terkait transaksi:
  - Create QRIS
  - Create Virtual Account
  - Check Transaction Status
  - Create Withdrawal
  - Create Withdrawal Account
- Menyempurnakan organisasi sidebar dan alur navigasi.
- Mendokumentasikan **Refresh Token** (`POST /api/v2/auth/refresh`).
